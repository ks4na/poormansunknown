## 站点配置文件

`docusaurus.config.js` 是整个站点的配置文件。

### 自定义配置字段

`docusaurus` 不允许使用配置文件中未知的字段，如果需要自定义的字段，需要在 `customFields` 字段中指定：

```sh
module.exports = {
  // ...
  customFields: {
    hello: 'world'
  }
}
```

## 组件中访问配置信息

配置文件对象可以在站点的任何组件中访问到，在 React 组件中使用如下代码获取：

```tsx {2,5}
import React from 'react'
import useDocusaurusContext from '@docusaurus/useDocusaurusContext'

const Hello = () => {
  const { siteConfig } = useDocusaurusContext()
  const { title, tagline } = siteConfig

  return <div>{`${title} · ${tagline}`}</div>
}
```
